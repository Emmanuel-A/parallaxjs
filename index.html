Parallax.js


Parallax Engine that reacts to the orientation of a smart device.

Why Parallax.js?


Parallax.js library uses no gyroscope or motion detection hardware. The position of the cursor is used instead.



Check it out!


For the best experience, check out this site on a mobile or tablet equipped with a gyroscope.

It's free!


Parallax.js library offers a simple api to get going with all your parallax needs. Also, it is open source and free. Learn how to use it today.
